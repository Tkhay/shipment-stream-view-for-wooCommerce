# WordPress.org Assets

This folder contains screenshots and graphics for the WordPress.org plugin directory.

## Required Images

### Screenshots

- `screenshot-1.png` - Customer-facing order tracker search form (1280x720px recommended)
- `screenshot-2.png` - Visual progress bar showing order status
- `screenshot-3.png` - Admin interface for managing tracking steps
- `screenshot-4.png` - Drag and drop to reorder tracking milestones
- `screenshot-5.png` - Exception alert display for special order statuses
- `screenshot-6.png` - Design settings for customization
- `screenshot-7.png` - Mobile responsive tracking display

### Plugin Icons (Optional but Recommended)

- `icon-128x128.png` - Plugin icon for WP admin
- `icon-256x256.png` - High-res plugin icon

### Banners (Optional but Recommended)

- `banner-772x250.png` - Plugin directory banner
- `banner-1544x500.png` - High-res banner (retina)

## Guidelines

- **Format**: PNG or JPG (PNG preferred)
- **Screenshot dimensions**: 1280x720px (16:9 ratio) recommended, max 1280px width
- **Icon dimensions**: Exactly 128x128 and 256x256
- **Banner dimensions**: Exactly 772x250 and 1544x500
- **Compression**: Optimize images to reduce file size
- **Naming**: Must match exactly (lowercase, hyphens)

## Note

When submitting to WordPress.org, these will go in the **SVN assets folder**, not in the plugin ZIP file itself.
