A modern, visual order tracking system for WooCommerce that displays order status with an elegant progress bar.

== Description ==

**Shipment Stream View for WooCommerce** transforms the way your customers track their orders with a beautiful, intuitive visual progress bar. No more confusing status labels – your customers will see exactly where their order is in the delivery process.

= Features =

* **Visual Progress Tracking** - Beautiful progress bar showing order journey
* **Customizable Status Steps** - Drag and drop to reorder your tracking milestones
* **Exception Handling** - Special alerts for order issues (on-hold, cancelled, etc.)
* **Easy Integration** - Simple shortcode `[ssvfw_order_tracker]` to add to any page
* **Responsive Design** - Perfect on desktop, tablet, and mobile devices
* **WooCommerce Compatible** - Works seamlessly with your existing WooCommerce setup
* **HPOS Ready** - Full support for High-Performance Order Storage
* **Design Customization** - Choose your brand colors and fonts
* **Milestone & Exception Types** - Categorize statuses as normal progress or special alerts

= How It Works =

1. Install and activate the plugin
2. Go to **Shipment Stream** in your WordPress admin
3. Customize your tracking steps (drag to reorder, edit labels, set types)
4. Add the `[ssvfw_order_tracker]` shortcode to any page
5. Customers enter their order ID and email to track their order

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to **Plugins → Add New**
3. Search for "Shipment Stream View for WooCommerce"
4. Click **Install Now** and then **Activate**

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Navigate to **Plugins → Add New → Upload Plugin**
4. Choose the ZIP file and click **Install Now**
5. Activate the plugin

= After Installation =

1. Go to **Shipment Stream** in your WordPress admin menu
2. Configure your tracking steps (milestones and exceptions)
3. Optionally customize design colors in **Settings**
4. Create a new page and add the shortcode: `[ssvfw_order_tracker]`
5. That's it! Your customers can now track their orders

== Frequently Asked Questions ==

= How do I add the order tracker to my site? =

Simply add the shortcode `[ssvfw_order_tracker]` to any page or post where you want the tracking form to appear.

= Can I customize the tracking steps? =

Yes! Go to **Shipment Stream** in your admin menu. You can drag and drop to reorder steps, edit labels, and categorize each status as either a "Milestone" (shown on the progress bar) or "Exception" (shown as an alert).

= What's the difference between Milestone and Exception? =

**Milestones** are normal order progression steps (like "Processing", "Shipped", "Delivered") shown on the visual progress bar. **Exceptions** are special situations (like "On Hold", "Cancelled") displayed as alert boxes to grab attention.

= Does it work with custom WooCommerce order statuses? =

Yes! The plugin automatically detects all your WooCommerce order statuses, including custom ones you've added.

= Is it compatible with High-Performance Order Storage (HPOS)? =

Absolutely! The plugin is fully compatible with WooCommerce's HPOS feature.

= Can I change the colors to match my brand? =

Yes! Go to **Shipment Stream → Settings** to customize the primary color and font family.

= What information do customers need to track their order? =

Customers need their Order ID and the email address used when placing the order. This ensures secure access to order information.

= Is the tracking page mobile-friendly? =

Yes! The tracking interface is fully responsive and works beautifully on all devices.
